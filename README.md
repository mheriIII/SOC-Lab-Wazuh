# SOC-Lab-Wazuh
Home SOC Lab: End-to-end SIEM deployment with Wazuh, Windows 11 telemetry, and Kali Linux attacks
This repository documents the design and deployment of a virtualized Security Operations Center (SOC) lab. The environment features a Wazuh SIEM manager (Ubuntu) used to ingest and analyze security telemetry from a Windows 11 endpoint.

To validate the lab's defensive capabilities, I performed various attack simulations—including SMB brute-force and network reconnaissance—using Kali Linux. All detected security events were successfully correlated and mapped to the MITRE ATT&CK framework to demonstrate a comprehensive understanding of the modern threat landscape.
